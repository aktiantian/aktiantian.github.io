##字典Dictionary
一种映射类型，无序。key:value。好处查询添加删除速度快，数据再多都很快。占用内存大。key要用不可变对象，不要是list等可变的。字典是哈希算法。key不能重复。

  • 定义方式
    原始对应 字典对应
  • 添加
  • 覆盖数据
  • 判断是否包含某数据
  • 取值的两种方式，推荐get，get没找到不会报错。
  • 删除pop。

##集合set
无序不重复的集合，与字典类似，但是只存储key，被用来求交集并集差集，去重复。

  • 创建两种定义方式， 会自动去重复

  • 删除
  • 交集 并集
